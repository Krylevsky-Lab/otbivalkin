Spaceship 2D Game Sprites

Description

We publish a set of Spaceship 2D Game Sprites. It includes 3 space fighters. Each of them has the possibility of improvement (adding new armor, weapons, etc.). Thus the appearance changes greatly. So if you want you can use them as different spaceships. We also created a simple animation of flight, shooting and explosion. All this is presented in PNG files. In the archive you will find the following formats: AI, EPS, PNG and SCML (animation created in the Spriter program). We wish you success in your game project!
