Pixel Art Space Trap Game Asset Pack

Description

Pixel Art Space Trap Game Asset Pack is a great collection of animated objects. All of them are designed to counteract the player: shoot, explode, be an obstacle or a trap. This set will be able to diversify the 2D space shooter. Graphics are created in Adobe Photoshop. In the archive you will find the following formats: PSD, PNG and SCML.

The kit contains:
  • Spike trap;
  • Mine with a rocket;
  • Kamikaze;
  • Meteorite;
  • Laser guns;
  • Mines;
  • Tank;
  • Spaceship.
